# 02-02: Anthropic Client

## Status: COMPLETE

## Files Created/Modified
- cli/llm/anthropic.go

## Notes
- Hand-rolled HTTP client with net/http, no external SDK.
- SSE parsing with bufio.Scanner, handles content_block_delta, message_delta, and message_start events.
- System prompt placed in top-level "system" field, not in messages array.
- Error mapping: 401 → auth, 429 → rate_limit (with Retry-After parsing), 5xx → network.
- Default max_tokens of 4096 when not specified.
